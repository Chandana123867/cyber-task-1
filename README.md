# Task 1 - Network Scan

## Objective
Scan local network and identify open ports.

## Tool Used
Nmap

## Steps
1. Installed Nmap
2. Found IP using ipconfig
3. Ran TCP SYN scan

## Results
Devices found:
- Router (192.168.31.1)
- Mobile (192.168.31.238)
- Laptop (192.168.31.198)

Open ports:
- 53 (DNS)
- 80 (HTTP)
- 443 (HTTPS)
- 135, 139, 445 (Windows services)

## Conclusion
Learned basic network scanning.
